A tag <pre>

HTML texto preformatado (<pre>) é a tag utilizada para representar texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado. A seguir, o código de como seria o texto acima sem a tag <pre>

<!DOCTYPE html>
<html>
    <head>
        <title>Estrutura de uma página web</title>
    </head>
    <body>
        <p>Conteúdo da página</p>
    </body>
</html>
